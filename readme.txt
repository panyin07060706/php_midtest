mid1.php為首頁
功能 - 按頁面的"GO!"即可進入主要頁面

trytry.php為校草頁面
功能 -
點每個人的"Learn More"可進入更詳細的資料簡介/按"Back"則可以回到上一頁
進入 簡介頁面 按"留言"則可以回覆參與討論本文章
進入 留言頁面 打上要回覆的內容後按"送出留言"即可確認留言
進入 留言確認頁面 按"新增留言" 即可再新留一則留言/按"檢視留言"即可回文章頁面查看留言

flower.php為校花頁面
功能 -
點每個人的"Learn More"可進入更詳細的資料簡介/按"Back"則可以回到上一頁
進入 簡介頁面 按"留言"則可以回覆參與討論本文章
進入 留言頁面 打上要回覆的內容後按"送出留言"即可確認留言
進入 留言確認頁面 按"新增留言" 即可再新留一則留言/按"檢視留言"即可回文章頁面查看留言

level.php為排行榜頁面
功能 -

register.php/register2.php為註冊頁面
功能 -
填完"基本資料"和"進階資料"即可按下"確認送出"
進入 資料確認頁面 檢查完畢後即可按"Back"回去登入頁面登入

login.php/login2.php為登入頁面 
功能 -
打上註冊過的帳號和密碼後 按下"登入" 即可進入系統
(如果帳號或密碼錯誤，則會3秒後從新進入登入頁面！)
